# AI Review

## What the AI generated
A `greet(name)` function in greet.py that returns a greeting, with an empty-name fallback.

## What I modified / verified
- Read the code line by line and confirmed the empty-name check works.
- Tested it by running `python greet.py` (printed the expected greeting).

## Why Git is essential when using AI
- Every AI-generated change is tracked, so I can see exactly what was added and revert if it breaks.
- Commit history separates "AI wrote this" from "I changed this," keeping accountability and traceability.
- PRs let AI output be reviewed before it reaches main — AI assists, but a human still owns the merge.
