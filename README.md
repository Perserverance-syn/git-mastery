# Git Mastery

A hands-on project completing 20 Git & GitHub challenges — from init to a full team-style workflow.

## What this demonstrates
- Staging, clean commits, history inspection
- Branching, merging, conflict resolution
- Stash, squash, cherry-pick
- Tags / releases (v1.0.0)
- GitHub: remote, push, Pull Requests
- Responsible AI-assisted development (see ai-review.md)

## How to run
1. Clone the repo
2. Read the challenge files
3. Follow the commit history to see each step
